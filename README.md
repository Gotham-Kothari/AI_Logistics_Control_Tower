## 🌟 Overview
ShipIt is an AI-assisted logistics platform that transforms fragmented shipment tracking into a unified, proactive decision system.

Instead of just telling you **what is happening**, ShipIt helps you decide **what to do next**.

## 🚀 Key Features
- **Unified Dashboard** – Real-time visibility across shipments and alerts  
- **Event-Driven Tracking** – Automatic updates via shipment events  
- **Risk & Exception Detection** – Early identification of issues  
- **AI Recommendations** – Context-aware suggestions using LLMs  
- **Human-in-the-Loop** – AI assists, operators decide  
- **Decision Logging** – Build institutional knowledge  

---

## 🧠 How It Works
```
Events → State Update → Risk Detection → Exceptions → AI Suggestions → Human Action
```

---

## 🏗️ Architecture
- **Frontend:** Dashboard (React / Flutter)  
- **Backend:** FastAPI (Python APIs)  
- **Intelligence Layer:** Rule-based + AI (LangChain, LangGraph)  
- **Database:** SQLite → PostgreSQL (scalable)
